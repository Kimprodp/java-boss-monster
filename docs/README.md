# 기능 목록
- 게임 시작 전 설정 단계
  - 보스 몬스터의 HP를 등록한다.
  - 플레이어의 이름과 HP를 등록한다.
- 게임 시작
  - 보스 몬스터와 플레이어의 현재 상태를 호출한다.
  - 플레이어는 공격방식을 선택하고 공격한다.
    - 공격 후 보스 몬스터의 상태를 확인하여 게임 종료 조건을 확인한다.
  - 보스 몬스터는 플레이어를 공격한다.
    - 공격 후 플레이어의 상태를 확인하여 게임 종료 조건을 확인한다.
  - 공격 작업의 대한 결과를 호출한다.
- 게임 종료
  - 보스 몬스터나 플레이어의 HP가 0일 경우 게임이 종료되며 턴이 실행되지 않는다.
  - 게임 종료에 대한 결과를 호출한다.
